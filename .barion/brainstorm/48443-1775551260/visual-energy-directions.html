"표면의 시각적 에너지" — 3가지 방향

레퍼런스 분석 기반. 구조(레이아웃)는 유지하고, 표면(질감)만 바꿉니다. 하네스에 범용 반영할 원칙을 도출하는 것이 목표.

A. 컬러 블록 전략

레퍼런스 핵심: 큰 면적에 짙은 색을 과감하게. 사이드바 / 히어로 카드 / 섹션 헤더에 다크 톤 적용.

매체 위치
바뀌는 것: 사이드바 다크, 히어로 카드 다크그린 배경, 숫자 white-on-dark
효과: 화면에 "무게감"이 생김. 색의 면적이 넓어져서 단조로움 탈출
리스크: B2B 도구에서 너무 튈 수 있음

B. 데이터 시각화 전략

레퍼런스 핵심: 숫자를 텍스트가 아닌 차트/바/도넛으로. 시각 요소의 다양성으로 단조로움 탈출.

리스크: 프로토타입 구현 비용 증가 (차트 라이브러리 or CSS)

C. A + B 혼합

컬러 블록(다크 사이드바 + 히어로) + 데이터 시각화(차트/도넛) 모두 적용. 레퍼런스에 가장 가까움.

하네스 반영 관점

어떤 방향을 선택하든, 하네스에 반영되는 원칙은:

  • anti-factory 원칙 6: "표면에 시각적 에너지가 있어야 한다" — 컬러 면적, 데이터 시각화, 대담한 블록 중 최소 하나
  • design-foundation 확장: "컬러 사용법"에 면적 가이드 추가 (어디에 큰 면적으로 쓸 것인가)
  • 화면 명세(GATE D1) 체크 추가: "이 화면에 숫자만 있는 카드가 몇 개인가? 3개 이상이면 시각화 대체 검토"
