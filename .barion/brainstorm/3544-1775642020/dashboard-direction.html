A. 어드민 Lite

같은 3열 구조, 시스템 상태 섹션만 제거. 지도 + 매체 상태 카드 + 헬스체크. 개발 빠름, 두 역할 일관성 높음.

장점

  • 어드민과 레이아웃 공유
  • 지도로 위치 파악 가능

단점

  • 동기화 반영 여부가 묻힘
  • 공장형 리스크 — 두 화면이 동일해 보임

B. 동기화 히어로 ★추천

다크 히어로 영역에 "편성 동기화 현황" 도넛 + 미반영 매체 목록. 지도는 보조. 김서영의 핵심 불안에 직결.

장점

  • 어드민과 시각적으로 명확히 구분
  • 3초 안에 "몇 대 반영됐나" 파악
  • 도메인 특화 UI (동기화 도넛)

단점

  • 어드민 대시보드와 다른 레이아웃 구현

C. 오늘의 운영 체크리스트

오늘 편성 현황 + 소재 검수 현황 + 동기화 상태 3열. 지도 없음. 하루 업무 시작 시 상황 파악 최적화.

장점

  • 매체사 업무 흐름에 맞춤
  • 소재 검수 현황까지 포함

단점

  • 지도 없음 — 위치 기반 파악 불가
  • 어드민과 가장 이질적 구조
