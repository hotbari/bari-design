A. 전체 합산 뷰 ★ 추천

GNB에 매체사 탭(전체/A사/B사/C사). 기본은 "전체" 집계 — 150대 통합 현황 + 매체사별 요약 행. 탭 클릭 시 해당 매체사 단독 뷰로 전환. 이준혁이 한눈에 전체를 파악하고 문제 매체사로 드릴다운.

B. 매체사 선택 후 단독 뷰

컨텐츠 상단 드롭다운으로 매체사를 선택하면, 해당 회사만 대시보드에 표시. 매체사 대시보드와 동일한 구조. 전체 집계 뷰 없음 — 한 번에 하나씩만 볼 수 있어 여러 이상 상황 비교가 어려움.

C. 멀티 컬럼 카드

3개 매체사를 나란히 카드로 배치. 이상 있는 회사는 테두리 강조. 지도 없음 (공간 부족). 카드 클릭 시 해당 매체사 상세로 드릴다운. 비교는 쉽지만 각 회사의 상세 정보가 적음.
