# UI Design

UI 디자인 스킬 허브.

## Available Commands

| Command | Description |
|---------|-------------|
| `/color-palette` | 풀 컬러 팔레트 생성 + 시맨틱 매핑 + A11y 검증 |
| `/design-screen` | 화면 레이아웃 설계 (구조, 계층, 타이포, 컬러, 스페이싱, 반응형, 다크모드) |
| `/type-system` | 타이포그래피 시스템 정의 |
| `/responsive-audit` | 반응형 디자인 검증 |

## Sub-Skills

| Skill | Description |
|-------|-------------|
| color-system | 컬러 토널 스케일 + 시맨틱 매핑 |
| dark-mode-design | 다크모드 컬러 전략 |
| data-visualization | 데이터 시각화 컬러·표현 |
| illustration-style | 일러스트레이션 스타일 |
| layout-grid | 그리드 시스템 |
| spacing-system | 스페이싱 스케일 |
| typography-scale | 타입 스케일·계층 |
| visual-hierarchy | 시각적 계층 구성 |

## Usage

인자 없이 실행 시 커맨드 목록을 표시하고 용도를 질문합니다.
