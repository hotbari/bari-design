# Design System

디자인 시스템 스킬 허브.

## Available Commands

| Command | Description |
|---------|-------------|
| `/audit-system` | 디자인 시스템 전반 품질 감사 |
| `/create-component` | 컴포넌트 스펙 작성 + 구현 |
| `/tokenize` | 디자인 토큰 정의 및 추출 |

## Sub-Skills

| Skill | Description |
|-------|-------------|
| accessibility-audit | 접근성 체계 점검 |
| component-spec | 컴포넌트 명세 문서화 |
| design-token | 디자인 토큰 시스템 |
| theming-system | 테마 전환 및 관리 |
| icon-system | 아이콘 라이브러리 |
| pattern-library | 재사용 패턴 문서화 |
| naming-convention | 네이밍 표준 |
| documentation-template | 문서 구조 템플릿 |

## Usage

인자 없이 실행 시 커맨드 목록을 표시하고 용도를 질문합니다.
