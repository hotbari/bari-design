## 2. 해소된 모순 (13건)

### C-1 [RESOLVED]: 대시보드 편성 위젯에 "일시정지" 상태 누락
- **결정**: 별도 항목으로 표시 (예약됨/적용중/일시정지/종료)
- **근거**: 4명 만장일치. 일시정지가 적용중에 묻히면 장애 감지 지연
- **반영**: 07-dashboard-report.md UC-29

### C-2 [RESOLVED]: BR ID 충돌
- **결정**: 도메인별 네임스페이스 (USR-BR, MED-BR, MAT-BR, CAM-BR, SCH-BR, NTF-BR)
- **근거**: 4명 만장일치. 운영 소통 시 도메인 즉시 특정 가능
- **반영**: 7개 파일 전체 BR 번호 재정리

### C-3 [RESOLVED → C-10으로 보완]: 캠페인 취소 시 편성표 영향
- **결정**: 캠페인 취소 시 연결 편성표 연쇄 종료 (캠페인-편성표 FK 연결된 경우에만)
- **근거**: 4명 만장일치. 취소 후 광고 계속 송출은 정산 사고
- **보완**: C-10에서 연결 모델 확정. "빈 구좌 필러 대체"는 미합의 사항으로 삭제
- **반영**: 04-campaign-management.md UC-19, 05-scheduling-management.md FSM 전이표

### C-4 [RESOLVED]: "예약됨" 편성표 + 매체 비활성화
- **결정**: 예약됨 상태 유지 + 알림 발송. 시작일 도래 시 매체 여전히 비활성이면 적용중 → 일시정지
- **근거**: 4명 만장일치. 매체 점검은 일시적인 경우가 대부분
- **반영**: 05-scheduling-management.md UC-25

### C-5 [RESOLVED]: 영업대행사 수동검수대기 알림
- **결정**: 소재 등록자에게도 정보성 알림 발송
- **근거**: 4명 만장일치. 등록자가 상태를 모르면 매체사에 문의 폭주
- **반영**: 06-notification-management.md 이벤트 카탈로그, 03-material-management.md UC-14

### C-6 [RESOLVED]: 매체사 삭제 시 소속 사용자 처리
- **결정**: 경고 표시 ("소속 사용자 N명 비활성화됩니다") 후 확인 시 일괄 비활성화
- **근거**: 4명 만장일치. A(가드만)는 번거롭고, B(무조건 자동)는 실수 복구 어려움
- **반영**: 02-media-management.md INV-05a

### C-7 [RESOLVED]: 배치 전환 시점
- **결정**: 편성표는 분 단위 즉시 전환, 캠페인은 매일 00:00(KST) 일 단위 배치
- **근거**: 4명 만장일치. 편성표는 "오후 2시부터 송출" 같은 정밀 제어 필요
- **반영**: 04-campaign-management.md CAM-BR-05, 05-scheduling-management.md SCH-BR-05

### C-8 [RESOLVED]: 소재 참조 단위 용어
- **결정**: "재생목록"으로 통일 (기술적으로 정확)
- **근거**: 어드민·운영대행사 2:2 매체사·영업대행사, 사용자 최종 결정 A
- **반영**: 03-material-management.md UC-16

### C-9 [RESOLVED]: 편성 동순위 충돌 규칙 모순
- **결정**: 동순위 편성표 저장 자체를 불허. 충돌 시 반드시 우선순위를 다르게 설정하여 해소
- **근거**: UC-22 "먼저 등록" vs UC-24 "나중에 생성"이 정반대 모순. 모호함 제거
- **반영**: 05-scheduling-management.md UC-22 (line 106), UC-24 (line 183) 수정

### C-10 [RESOLVED]: 캠페인↔편성표 연결 모델
- **결정**: 편성표에 캠페인 FK 선택 필드 추가. 연결 시에만 캠페인 취소 연쇄 종료 작동. 미연결 편성표는 영향 없음
- **근거**: 캠페인 연동은 필수가 아님. 독립 편성도 가능해야 함
- **반영**: 05-scheduling-management.md UC-22 step 6 추가, FSM 전이표에 캠페인 취소 트리거 추가. 04-campaign-management.md 미결 사항 해소

### C-11 [RESOLVED]: "필러" 개념 이중 사용
- **결정**: UC-19 "빈 구좌는 기본 소재(필러)로 대체"는 미합의 사항으로 삭제. 캠페인 유형 "필러"만 유효 (회의 2 합의)
- **근거**: 빈 구좌 대체 맥락은 논의된 적 없는 문서 오기재
- **반영**: 04-campaign-management.md UC-19 line 127 수정

### C-12 [RESOLVED]: "매체 관리자" 미정의 역할
- **결정**: "매체 관리자" = 해당 매체의 매체사 + 운영대행사
- **근거**: 기존 4개 페르소나에 없는 역할. 구체적 수신 대상으로 대체
- **반영**: 06-notification-management.md 이벤트 카탈로그 "매체 관리자" → "해당 매체의 매체사 + 운영대행사"

### C-13 [RESOLVED]: 소재 운영 기간 만료 — 편성 연결 가드
- **결정**: 만료 시 경고만. 편성 연결은 허용하되 "운영 기간 만료 소재입니다" 경고 표시
- **근거**: 운영 기간은 참고 정보. 재계약 등으로 만료 소재를 의도적으로 사용하는 케이스 허용
- **반영**: 03-material-management.md INV-11 보완

## 4. 교차 엔티티 의존성

### 매체 → 편성표 (강한 의존)
- 매체 비활성화 → 적용중 편성표 일시정지 (원자적 트랜잭션)
- 매체 활성화 → 종료일 재검사 후 복귀/종료

### 소재 → 구좌 (강한 의존)
- 소재 삭제 → 구좌 "삭제됨" (재생 시 스킵)
- 소재 교체 → 구좌 "교체중" (기존 파일 유지)

### 캠페인 → 편성표 (선택적 의존, C-3 + C-10 + C-14으로 확정)
- 편성표에 캠페인 FK 선택 연결
- 캠페인 취소 → 연결된 예약됨/적용중 편성표 연쇄 종료
- 캠페인 완료 → 편성표 자동 종료 안 함. 적용중 편성표 있으면 알림만 발송 (C-14)
- 미연결 편성표는 캠페인 취소/완료 영향 없음

### 소재 → 편성표 (INV-11 게이트)
- 검수완료 소재만 재생목록 구좌 배치 가능
- 운영 기간 만료 소재는 경고 후 연결 허용 (C-13)
