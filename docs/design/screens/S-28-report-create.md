# S-28: 리포트 생성

## 출처
- UC-30: 리포트 생성 및 예약 발송

## 레이아웃
- form-card 기반

## 컴포넌트
- form-card "리포트 설정"
  - 유형 select
  - 대상 매체 다중 선택
  - 기간 date range
- form-card "예약 발송"
  - 예약 토글
  - 주기 select (매일 / 매주 / 매월)
  - 수신자 이메일 태그 입력
  - 발송 시각
- 하단 액션
  - btn-primary "즉시 생성"
  - btn-secondary "예약 저장"
  - btn-ghost "취소"

## 상태
- **기본**: 폼 입력 대기
- **예약 토글 OFF**: 예약 발송 섹션 숨김
- **예약 토글 ON**: 예약 발송 섹션 표시
- **생성 중**: 버튼 로딩
- **완료**: 성공 토스트 + 목록 이동

## 접근성
- 예약 토글 변경 시 aria-expanded로 섹션 표시 상태 전달
- 이메일 태그 키보드 조작 (Enter 추가, Backspace 삭제)
- 필수 필드 aria-required

## 역할 분기
- 어드민: 전체 매체 대상 생성
- 매체사: 전체 매체 대상 생성
- 운영대행사: 부여된 매체만 대상 선택 가능
