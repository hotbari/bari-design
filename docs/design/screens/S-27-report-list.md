# S-27: 리포트 목록

## 출처
- UC-31: 리포트 조회 / 다운로드

## 레이아웃
- 표준 목록 (filter-bar + data-table)

## 컴포넌트
- filter-bar
  - 유형 select
  - 기간 date range
  - 매체 select
- data-table
  - 리포트명
  - 유형
  - 기간
  - 생성일
  - 상태 badge (생성중 / 완료 / 실패)
  - 다운로드 아이콘 버튼
- drawer 미리보기

## 상태
- **기본**: 리포트 목록 표시
- **생성중**: badge "생성중" + 다운로드 비활성
- **완료**: badge "완료" + 다운로드 활성
- **실패**: badge "실패" + 재생성 안내
- **drawer 열림**: 리포트 미리보기

## 접근성
- 다운로드 버튼 aria-label에 리포트명 포함
- badge 상태 스크린리더 전달
- drawer 포커스 트랩

## 역할 분기
- 어드민: 전체 리포트 조회
- 매체사: 전체 리포트 조회
- 운영대행사: 부여된 매체 리포트만 조회
