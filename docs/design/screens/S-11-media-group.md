# S-11: 매체 그룹 관리

## 출처
- UC-37: 그룹 CRUD, 매체 배정

## 레이아웃
- 2패널 구조
  - 좌: 그룹 목록
  - 우: 그룹 상세 / 매체 배정

## 컴포넌트
- 좌측 패널
  - 그룹 카드형 목록 (이름 + 소속 매체 수)
  - btn-primary "그룹 추가"
- 우측 패널
  - 그룹명 인라인 편집
  - 배정 매체 체크박스 목록
  - 미배정 매체 목록
  - btn-destructive "그룹 삭제"

## 상태
- **기본**: 그룹 목록 표시, 첫 번째 그룹 선택
- **그룹 추가**: 좌측에 이름 입력 인라인 폼
- **그룹 삭제 확인**: 확인 다이얼로그

## 접근성
- 좌측 목록 keyboard navigation
- 그룹 삭제 시 확인 다이얼로그 포커스 트랩
