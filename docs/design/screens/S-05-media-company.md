# S-05: 매체사 관리

## 출처
- UC-06: 어드민이 매체사 등록

## 레이아웃
- 표준 CRUD (page-header + data-table)
- 등록/수정: drawer

## 컴포넌트
- page-header "매체사 관리" + btn-primary "매체사 등록"
- data-table
  - 매체사명
  - 사업자등록번호
  - 대표자
  - 소속 매체 수
  - 등록일
- drawer (등록/수정)
  - 매체사명
  - 사업자등록번호 (패턴: 000-00-00000)
  - 대표자명
  - 연락처
  - 주소

## 상태
- **목록**: 기본 테이블 뷰
- **drawer 열림**: 등록 또는 수정 폼
- **사업자번호 중복 에러**: 필드 에러 + 메시지

## 접근성
- drawer 열림 시 포커스 트랩
- 사업자등록번호 입력 패턴 안내
