# S-01: 가입 완료

## 출처
- UC-02: 초대받은 사용자가 가입 완료
- 초대 기반 전용 (공개 가입 없음)

## 레이아웃
- 중앙 정렬 카드
- 상단: 초대 정보 (초대한 사람, 부여된 역할)

## 컴포넌트
- 이름 입력 (required)
- 비밀번호 입력 (required)
- 비밀번호 확인 입력
- 비밀번호 강도 표시 바
- "가입 완료" btn-primary

## 상태
- **기본**: 폼 입력 대기
- **비밀번호 불일치**: 확인 필드 에러 표시
- **토큰 만료**: 에러 카드 + 재발급 안내
- **완료**: 성공 메시지 + 로그인 페이지 링크

## 접근성
- autofocus on 이름 필드
- 비밀번호 강도 바 aria-live="polite"

## 역할 분기
- 전체 동일 (인증 전 화면, 초대 링크 기반)
