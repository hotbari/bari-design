# 대시보드 — 운영대행사 (`dashboard-ops-agency.html`)

## 기본 정보

- **페이지 제목**: 대시보드
- **역할**: 운영대행사 (ops)
- **레이아웃**: 사이드바(240px) + 메인 (GNB + 바디)

## 레이아웃 구조

- GNB: 대시보드 제목 + 컨텍스트 스위처(탭) + 날짜 + 알림벨
- 바디: 좌측 스크롤 패널(65%) + 우측 패널(35%)

## GNB 컨텍스트 스위처

- 담당 매체사 목록을 탭으로 전환 (최대 N개)
- `.ctx-switcher` 컨테이너에 `.ctx-tab` 항목
- 선택된 탭: `.ctx-tab.active`

## 좌측 패널 섹션

| 섹션 | 내용 |
|------|------|
| 폴링 오류 배너 | 데이터 조회 오류 시 경고 (warning 스타일) + 재시도 버튼 |
| 이상 배너 | 이상 감지 이슈 (error 스타일) + 링크 |
| 업무 큐 히어로 (Task Queue) | 4컬럼 그리드: 검수대기/편성충돌/동기화오류/완료 건수 |
| 매체사별 현황 | 담당 매체사 매체 목록 + 상태 |

## 업무 큐 카드

| 항목 | CSS 클래스 | 색상 |
|------|-----------|------|
| 검수대기 | task-cell.urgent | red (task-count.red) |
| 편성충돌 | task-cell.warn | yellow (task-count.yellow) |
| 동기화오류 | task-cell.warn | yellow |
| 완료 | task-cell.good | green (task-count.green) |

## 기타 UI

- **컨텍스트 스위처**: 담당 매체사 탭 전환으로 해당 매체사 데이터만 표시
- **업무 큐**: 클릭 가능한 카드, 각 항목 상세로 이동
