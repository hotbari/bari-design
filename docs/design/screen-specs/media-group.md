# 매체 그룹 (`media-group.html`)

## 기본 정보

- **페이지 제목**: 매체 그룹
- **브레드크럼**: 매체 관리 › 매체 그룹
- **레이아웃**: 사이드바(200px) + 메인 (GNB + 2패널)

## 레이아웃 구조

- GNB: 브레드크럼 + 우측 날짜 + 알림벨
- 바디: 좌측 패널 (그룹 목록) + 우측 패널 (그룹 상세)

## 좌측 패널: 그룹 목록

- 그룹 항목 목록 (선택 시 하이라이트)
- 인라인 추가 폼: 입력 후 Enter로 저장
- 새 그룹 추가 버튼

## 우측 패널: 그룹 상세

| 섹션 | 내용 |
|------|------|
| 그룹명 | 인라인 편집 가능 (input) |
| 배정된 매체 | 체크박스 목록, 상태 dot (online/error/delayed) |
| 미배정 매체 | 체크박스 목록, 상태 dot |

## 상태값/배지

```
// 매체 상태 dot
dot: 'online' | 'error' | 'delayed'
```

## 기타 UI

- **저장 버튼**: 그룹명/매체 변경 저장
- **그룹 삭제**: 삭제 버튼 → 확인 다이얼로그
- **매체 배정**: 미배정에서 체크박스 체크 → 배정됨으로 이동 (또는 저장 시 반영)
