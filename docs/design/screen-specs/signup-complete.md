# 가입 완료 (`signup-complete.html`)

## 기본 정보

- **페이지 제목**: 가입 완료
- **레이아웃**: 풀스크린 중앙 정렬 (사이드바 없음)

## 레이아웃 구조

- 배경: neutral-50
- 중앙 카드 (max-width 420px, padding 40px)

## 폼 필드

| 필드 | 타입 | 필수 | 비고 |
|------|------|------|------|
| 이름 | text | * | |
| 비밀번호 | password | * | 강도 표시 바 포함 |
| 비밀번호 확인 | password | * | |

## 기타 UI

- **브랜드**: Bari CMS 로고 마크 + 이름 (카드 상단)
- **초대 블록**: 초대한 조직/역할 정보 표시 (primary-50 배경)
- **비밀번호 강도 바**: 입력에 따라 실시간 강도 표시
- **가입 완료 버튼**: primary, full-width
- **Skip 링크**: 접근성용 skip-link
- **완료 후**: 로그인 페이지(login.html)로 이동
