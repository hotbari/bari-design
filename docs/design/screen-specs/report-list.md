# 리포트 목록 (`report-list.html`)

## 기본 정보

- **페이지 제목**: 리포트
- **브레드크럼**: 리포트 › 리포트 목록
- **레이아웃**: 사이드바(200px) + 메인 (GNB + 컨텐츠)

## 레이아웃 구조

- GNB: 브레드크럼 + 날짜 + 알림벨
- 컨텐츠: 페이지 헤더 → 필터바 → 툴바(카운터) → 테이블

## 테이블 컬럼

| # | 헤더 | 표시 데이터 | 비고 |
|---|------|------------|------|
| 1 | 리포트명 | 리포트명 (font-weight:600) | max-width:220px, ellipsis |
| 2 | 유형 | 유형 텍스트 | 캠페인 성과 / 매체 상태 / 편성 현황 |
| 3 | 기간 | 날짜 범위 | xs |
| 4 | 생성일 | 날짜 | xs |
| 5 | 상태 | 상태 배지 | generating/done/fail |
| 6 | (액션) | 다운로드 아이콘 (done만 활성) / 재시도 버튼 (fail) | hover 표시 |

## 필터

| 이름 | 타입 | 옵션 (value → 표시명) |
|------|------|----------------------|
| 유형 | select | `''`→전체, `campaign`→캠페인 성과, `media`→매체 상태, `schedule`→편성 현황 |
| 기간 | date-range | 시작일 ~ 종료일 |
| 매체 | select | `''`→전체 매체, `lotteworld`→롯데월드 타워, `coex`→코엑스 아트리움, `gangnam`→강남역 빌보드 |

초기화 버튼 있음

## 상태값/배지

```
// 리포트 상태
status: 'generating' | 'done' | 'fail'
// CSS: .badge-generating, .badge-done, .badge-fail
```

## 기타 UI

- **카운터**: 있음 ("총 N개 리포트")
- **주요 액션**: "리포트 생성" 버튼 (→ report-create.html)
- **행 클릭**: 우측 Drawer로 리포트 미리보기 (메타 그리드 + 차트 플레이스홀더)
- **다운로드**: done 상태 행에만 활성, fail 상태에는 "재시도" 버튼 표시
