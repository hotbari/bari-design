# 긴급 편성 (`emergency-schedule.html`)

## 기본 정보

- **페이지 제목**: 긴급 편성
- **브레드크럼**: 편성 관리 › 긴급 편성
- **레이아웃**: 사이드바(200px) + 메인 (GNB + 컨텐츠)
- **접근 권한**: 어드민 전용 (admin-only 배지 표시)

## 레이아웃 구조

- GNB: 브레드크럼 + 알림벨
- 컨텐츠:
  - 긴급 경고 배너 (red, 상단 고정)
  - 현재 활성 긴급 편성 목록
  - 긴급 편성 등록 폼

## 현재 긴급 편성 목록

| # | 항목 | 비고 |
|---|------|------|
| 상태 | em-active / em-inactive | CSS: .em-active, .em-inactive |
| 매체명 | 대상 매체 | |
| 소재명 | 적용 소재 | |
| 기간 | 집행 기간 | |
| 사유 | 긴급 사유 | |

## 폼 필드 (긴급 편성 등록)

| 필드 | 타입 | 필수 | 옵션/비고 |
|------|------|------|----------|
| 대상 매체 | checkbox list | * | 다중 선택 |
| 소재 | select | * | 소재 선택 |
| 집행 기간 | radio (duration options) | * | 1시간 / 2시간 / 4시간 / 직접 입력 |
| 사유 | textarea | * | 긴급 사유 입력 |
| 확인 체크박스 | checkbox | * | "위 내용을 확인했습니다" — 체크 전까지 실행 버튼 비활성화 |

## 상태값/배지

```
// 긴급 편성 상태
em_status: 'em-active' | 'em-inactive'
// CSS: .em-active, .em-inactive
```

## 기타 UI

- **긴급 경고 배너**: 빨간색 배너, 긴급 편성 위험성 안내
- **실행 버튼**: "긴급 편성 실행" (danger/red), 확인 체크박스 체크 전 disabled
- **어드민 전용 배지**: 페이지 헤더에 표시
