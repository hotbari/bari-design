# 대시보드 — 매체사 (`dashboard-media-company.html`)

## 기본 정보

- **페이지 제목**: 대시보드
- **역할**: 매체사 (media)
- **레이아웃**: 사이드바(240px) + 메인 (GNB + 바디)

## 레이아웃 구조

- GNB: 대시보드 제목 + 날짜 + 새로고침 버튼 + 알림벨
- 바디: 좌측 스크롤 패널(65%) + 우측 패널(35%)

## 좌측 패널 섹션

| 섹션 | 내용 |
|------|------|
| 폴링 오류 배너 | 데이터 조회 오류 시 경고 (warning 스타일) + 재시도 버튼 |
| 이상 배너 | 이상 감지 시 error 스타일, 이슈별 링크 |
| 동기화 히어로 | 도넛 차트 (동기화 비율) + 헤드라인 + 메타 정보 |
| 매체사별 현황 | 소속 매체사 목록 + 매체 수 + 상태 요약 |
| 헬스체크 이상 | 이상 있는 매체 목록 |

## 우측 패널

- 매체 상태 요약 카드
- 최근 편성 현황

## 특이사항

- **동기화 히어로**: 도넛 차트 (동기화율 %) + 동기화 완료/미완료/지연 수치
- **매체사별 현황**: 매체사 단위로 상태 요약 (각 매체사별 매체 수, 온라인/오프라인 비율)
