# 재생목록 편집 (`playlist-edit.html`)

## 기본 정보

- **페이지 제목**: 재생목록 편집
- **브레드크럼**: 편성 관리 › 재생목록 › 편집
- **레이아웃**: 사이드바(200px) + 메인 (GNB + 2패널)

## 레이아웃 구조

- GNB: 재생목록명 입력 + 즉시 적용 버튼 + 저장/취소
- 바디: 좌측 슬롯 캔버스 + 우측 소재 피커

## 좌측: 슬롯 캔버스

각 슬롯 항목 구성:
| 요소 | 설명 |
|------|------|
| 드래그 핸들 | drag-and-drop 순서 변경 |
| 순서 번호 | 슬롯 번호 |
| 썸네일 | 소재 썸네일 (36×64px) |
| 소재명 | font-weight:600 |
| 재생시간 | xs, neutral-500 |
| 액션 | 교체 / 삭제 버튼 |

슬롯 타입:
- `normal`: 일반 소재
- `dummy`: 더미 슬롯 (placeholder)
- `deleted`: 삭제된 소재 (경고 표시)

## 우측: 소재 피커

- 검색 입력 (소재 검색)
- 소재 목록 (클릭하여 슬롯에 추가)
- 더미 추가 버튼

## 기타 UI

- **드래그앤드롭**: 슬롯 순서 변경
- **즉시 적용**: 저장하지 않고 현재 편성에 즉시 반영
- **액션 버튼 (GNB)**: 저장 / 취소
- **삭제된 소재**: 슬롯에 `deleted` 타입으로 표시 (경고 스타일)
