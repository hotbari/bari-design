# 매체사 관리 (`media-company-mgmt.html`)

## 기본 정보

- **페이지 제목**: 매체사 관리
- **브레드크럼**: 매체 관리 › 매체사 관리
- **레이아웃**: 사이드바(200px) + 메인 (GNB + 컨텐츠)

## 레이아웃 구조

- GNB: 브레드크럼 + 우측 날짜 + 알림벨
- 컨텐츠: 페이지 헤더 → 검색바 → 툴바(카운터) → 테이블

## 테이블 컬럼

| # | 헤더 | 표시 데이터 | 보조 텍스트 | 비고 |
|---|------|------------|------------|------|
| 1 | 매체사명 | 매체사명 (font-weight:600) | | |
| 2 | 사업자등록번호 | 사업자등록번호 | | |
| 3 | 대표자 | 대표자명 | | |
| 4 | 소속 매체 | 매체 수 (count badge) | | |
| 5 | 등록일 | 날짜 | | xs |
| 6 | 액션 | 수정 버튼 | | hover on row |

## 기타 UI

- **카운터**: 있음 ("총 5개")
- **주요 액션**: "+ 매체사 등록" 버튼 → Drawer(우측 슬라이드) 열림
- **수정**: 행의 수정 버튼 → 같은 Drawer 수정 모드
- **중복 검증**: 사업자등록번호 중복 시 인라인 에러 표시
- **Drawer**: 우측에서 슬라이드인, 백드롭 클릭으로 닫힘
