# 편성표 등록/수정 (`schedule-form.html`)

## 기본 정보

- **페이지 제목**: 새 편성표 / 편성표 수정
- **브레드크럼**: 편성 관리 › 편성표 › 새 편성표
- **레이아웃**: 사이드바(200px) + 메인 (GNB + 2컬럼)

## 레이아웃 구조

- GNB: 브레드크럼
- 바디: 좌측 메인 폼 + 우측 미리보기 패널

## 폼 필드

### 섹션 1: 기본 설정

| 필드 | 타입 | 필수 | 옵션/비고 |
|------|------|------|----------|
| 편성표명 | text | * | placeholder: "편성표 이름 입력" |
| 우선순위 | radio cards | * | `1순위` / `2순위` / `3순위` |

### 섹션 2: 콘텐츠 및 매체

| 필드 | 타입 | 필수 | 비고 |
|------|------|------|------|
| 재생목록 | picker (검색 + 선택) | * | 재생목록 검색 후 선택 |
| 대상 매체 | checkbox list (+ 검색) | * | 매체 검색, 다중 선택 |

### 섹션 3: 적용 기간

| 필드 | 타입 | 필수 | 비고 |
|------|------|------|------|
| 시작일시 | datetime (date + time inputs) | - | |
| 종료일시 | datetime (date + time inputs) | - | |

## 기타 UI

- **충돌 감지 패널**: 기간 겹침 발생 시 충돌 목록 표시, 해결 옵션 제공
- **우측 미리보기**: 입력 내용 실시간 요약 표시
- **액션 버튼**: 저장 (primary) / 취소
