# 매체 등록/수정 (`media-form.html`)

## 기본 정보

- **페이지 제목**: 매체 등록 (신규) / 매체 수정 (수정)
- **브레드크럼**: 매체 관리 › 매체 등록
- **레이아웃**: 사이드바(200px) + 메인 (GNB + 컨텐츠, max-width 800px)

## 레이아웃 구조

- GNB: 브레드크럼
- 컨텐츠: 3개 카드 (기본 정보 / 디스플레이 설정 / 운영 시간) + 하단 액션 버튼

## 폼 필드

### 카드 1: 기본 정보

| 필드 | 타입 | 필수 | 옵션/비고 |
|------|------|------|----------|
| 매체명 | text | * | |
| 매체사 | select | * | 매체사 선택 |
| 매체 종류 | radio | * | `고정형` / `이동형` |
| 설치 주소 | text | * | |
| 설치 형태 | radio | * | `가로형` / `세로형` |
| 비고 | textarea | - | |

### 카드 2: 디스플레이 설정

| 필드 | 타입 | 필수 | 옵션/비고 |
|------|------|------|----------|
| 디스플레이 유형 | select | * | `LCD` / `LED` / `기타` |
| 화면 사이즈 | text | * | 예: 75인치 |
| 해상도 | select | * | 해상도 선택 (예: 1920×1080) |
| LED Pitch | text | 조건부* | 디스플레이 유형이 LED일 때만 표시 |
| 비정형 사이즈 | radio | 조건부 | LED일 때: `없음` / `있음` |

### 카드 3: 운영 시간

| 필드 | 타입 | 필수 | 옵션/비고 |
|------|------|------|----------|
| 운영 요일 | radio | * | `매일` / `평일` / `주말` / `직접 선택` |
| 운영 시간 | radio + 조건부 time inputs | * | `24시간` / `직접 입력` → 시작시간~종료시간 |

## 기타 UI

- **액션 버튼**: 등록 (primary) / 임시 저장 (secondary) / 취소
- **수정 모드**: URL query `?edit=1`로 진입
