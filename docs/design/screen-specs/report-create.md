# 리포트 생성 (`report-create.html`)

## 기본 정보

- **페이지 제목**: 리포트 생성
- **브레드크럼**: 리포트 › 리포트 생성
- **레이아웃**: 사이드바(200px) + 메인 (GNB + 컨텐츠, max-width)

## 레이아웃 구조

- GNB: 브레드크럼 + 날짜 + 알림벨
- 컨텐츠: 페이지 헤더 → 폼 카드

## 폼 필드

| 필드 | 타입 | 필수 | 비고 |
|------|------|------|------|
| 리포트명 | text | * | placeholder: "리포트명 입력" |
| 기간 | date-range | * | 시작일 ~ 종료일 |
| 대상 매체 | multi-select grid | - | 전체선택 버튼 포함, 체크박스 그리드 |
| 정기 발송 | toggle | - | on/off 토글 |
| 발송 주기 | select | 조건부 | 정기 발송 ON일 때: 주간/월간 등 |
| 발송 요일 | select | 조건부 | 정기 발송 ON일 때 |
| 발송 시간 | select | 조건부 | 정기 발송 ON일 때 |
| 수신자 이메일 | email tag input | - | 이메일 태그 입력 (여러 개) |

## 기타 UI

- **대상 매체 전체선택**: "전체 선택" 버튼으로 일괄 선택/해제
- **정기 발송 토글**: ON 시 발송 주기/요일/시간 필드 조건부 표시
- **이메일 태그**: 이메일 입력 후 Enter/콤마로 태그 추가
- **생성 버튼**: 클릭 시 loading 상태 (btn-loading)
- **액션 버튼**: 생성 (primary, with loading state) / 취소
