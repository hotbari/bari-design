# 소재 규격 안내 (`material-spec-guide.html`)

## 기본 정보

- **페이지 제목**: 소재 규격 안내
- **브레드크럼**: 소재 관리 › 소재 규격 안내
- **레이아웃**: 사이드바(200px) + 메인 (GNB + 컨텐츠)

## 레이아웃 구조

- GNB: 브레드크럼 + 날짜 + 알림벨
- 컨텐츠: 페이지 헤더 → 툴바(검색 + 매체사 필터) → 규격 카드 그리드

## 필터

| 이름 | 타입 | 비고 |
|------|------|------|
| 검색 | search | 매체명 검색 (toolbar 내) |
| 매체사 | select | 매체사 필터 |

## 카드 그리드

- 레이아웃: `grid-template-columns: repeat(auto-fill, minmax(340px, 1fr))`
- 각 카드 구성:
  - 매체명 (font-weight:600)
  - 매체사명 (xs)
  - 유형 배지 (type badge)
  - 규격 목록 (dt/dd 쌍): 해상도, 파일 형식, 최대 용량, 재생시간 등
  - 파일 형식 태그 (format tags)
  - 안내 텍스트

## 기타 UI

- **카드 레이아웃**: 반응형 그리드 (auto-fill)
- **카드 내 규격**: dl/dt/dd 형식의 규격 목록
