# 싱크 송출 설정 (`sync-schedule.html`)

## 기본 정보

- **페이지 제목**: 싱크 송출 설정
- **브레드크럼**: 편성 관리 › 싱크 송출
- **레이아웃**: 사이드바(200px) + 메인 (GNB + 2컬럼)

## 레이아웃 구조

- GNB: 브레드크럼 + 알림벨
- 바디: 좌측 폼 + 우측 타임라인 미리보기 패널

## 폼 필드

| 섹션 | 필드 | 타입 | 필수 | 비고 |
|------|------|------|------|------|
| 대상 매체 | 매체 선택 | chips + dropdown | * | 선택된 매체 chip으로 표시 |
| 소재 | 소재 선택 | card selector | * | 소재 카드 목록에서 선택 |
| 시간 스케줄 | 시간대별 슬롯 | time grid (+/- 슬롯) | * | 시간대별 재생 횟수 설정 |
| 반복 요일 | 요일 토글 | toggle buttons | * | 월/화/수/목/금/토/일 개별 선택 |

## 기타 UI

- **우측 타임라인**: 설정한 시간 스케줄 시각화
- **슬롯 제어**: 각 시간대 +/- 버튼으로 재생 횟수 조정
- **요일 토글**: 7개 요일 개별 on/off 토글 버튼
- **액션 버튼**: 저장 / 취소
