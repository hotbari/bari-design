# 사용자 초대 (`user-invite.html`)

## 기본 정보

- **페이지 제목**: 사용자 초대
- **브레드크럼**: 사용자 관리 › 사용자 초대
- **레이아웃**: 사이드바(200px) + 메인 (GNB + 2컬럼)

## 레이아웃 구조

- GNB: 브레드크럼 + 알림벨
- 바디: 좌측 초대 폼 + 우측 역할 안내 패널
- 성공 배너: 초대 발송 완료 시 상단에 슬라이드다운으로 표시

## 폼 필드

### 카드 1: 초대 정보

| 필드 | 타입 | 필수 | 옵션/비고 |
|------|------|------|----------|
| 이메일 주소 | email | * | placeholder: "example@company.com", 형식 유효성 검증 |
| 역할 | radio cards (2×2 그리드) | * | `media`→매체사, `ops`→운영대행사, `sales`→영업대행사, `admin`→어드민 |
| 소속 매체사 | select | * (비어드민) | 어드민 역할 선택 시 숨김, options: 한빛미디어/서울옥외광고/디지털사이니지 |

### 카드 2: 권한 범위 설정

| 역할 | 표시 내용 |
|------|----------|
| ops (운영대행사) | 접근 가능 매체/그룹 체크박스 목록 (미선택 시 전체 접근) |
| sales (영업대행사) | 소재 접근 범위 라디오: `all`→전체 소재 / `assigned`→배정된 소재만 |
| media (매체사) | 섹션 숨김 |
| admin (어드민) | 섹션 숨김 |

## 기타 UI

- **역할별 안내 패널**: 우측에 각 역할의 권한 범위 요약 표시
- **소속 매체사 없음 경고**: 선택한 매체사에 등록된 매체가 없을 경우 경고 표시
- **성공 배너**: 초대 완료 시 애니메이션으로 표시, "추가 초대" 버튼 포함
- **초대 유효기간**: 7일 (페이지 부제목에 명시)
- **폼 액션**: 초대 발송 (primary) / 취소
