# 캠페인 등록/수정 (`campaign-form.html`)

## 기본 정보

- **페이지 제목**: 캠페인 등록 (신규) / 캠페인 수정 (수정)
- **브레드크럼**: 캠페인 관리 › 캠페인 등록
- **레이아웃**: 사이드바(200px) + 메인 (GNB + 2컬럼 컨텐츠)

## 레이아웃 구조

- GNB: 브레드크럼
- 메인: 좌측 스텝 폼 + 우측 작성 요약 패널
- 3단계 스텝 위자드 (상단 step indicator)

## 폼 필드

### Step 1 — 기본 정보

| 필드 | 타입 | 필수 | 옵션/비고 |
|------|------|------|----------|
| 캠페인명 | text input | * | placeholder: "캠페인명을 입력하세요" |
| 광고주 | select + 신규 등록 버튼 | * | 광고주 선택 / "신규 등록" → 광고주 등록 모달 |
| 광고대행사 | text input | - | 선택 입력 |
| 캠페인 유형 | radio cards | * | `direct`→직접판매, `own`→자사광고, `filler`→필러, `naver`→네이버 |

### Step 2 — 매체 및 기간

| 필드 | 타입 | 필수 | 비고 |
|------|------|------|------|
| 대상 매체 | checkbox list (+ 검색) | * | 매체 검색 가능, 다중 선택 |
| 집행 기간 | date-range | * | 시작일 ~ 종료일, 충돌 경고 패널 |

### Step 3 — 예산 및 확인

| 필드 | 타입 | 필수 | 옵션/비고 |
|------|------|------|----------|
| 과금 모델 | select | * | `fixed`→정액제, `daily`→일단가, `none`→입력안함 |
| 집행 금액 | number input | - | 과금 모델이 fixed/daily일 때만 표시 |

### 광고주 신규 등록 모달

| 필드 | 타입 | 필수 |
|------|------|------|
| 회사명 | text | * |
| 사업자등록번호 | text | * |
| 담당자명 | text | - |
| 연락처 | tel | - |

## 기타 UI

- **우측 작성 요약**: 캠페인명, 광고주, 유형, 매체, 기간, 예산 실시간 표시
- **충돌 경고**: 기간 겹침 발생 시 해결 옵션 패널 표시
- **스텝 이동**: 이전/다음 버튼, 3단계에서 "등록" 버튼
- **임시 저장**: 없음 (campaign-form에는 없음)
