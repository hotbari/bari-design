# 내 정보 (`my-profile.html`)

## 기본 정보

- **페이지 제목**: 내 정보
- **브레드크럼**: 내 정보
- **레이아웃**: 사이드바(200px) + 메인 (GNB + 설정 레이아웃)

## 레이아웃 구조

- GNB: 브레드크럼 + 알림벨
- 바디: 좌측 설정 사이드내비(200px) + 우측 컨텐츠

## 설정 사이드내비

| 그룹 | 항목 |
|------|------|
| 내 계정 | 프로필 (active) / 알림 설정 / 보안 |
| 시스템 (어드민) | 시스템 설정 |

## 폼 필드

### 기본 정보 카드

| 필드 | 타입 | 필수 | 비고 |
|------|------|------|------|
| 아바타 | 이미지 (사진 변경 버튼) | - | JPG/PNG, 최대 2MB, 권장 200×200px |
| 이름 | text | - | 수정 가능 |
| 역할 | readonly field | - | 역할 배지로 표시, 수정 불가 |
| 이메일 | email (disabled) | - | 수정 불가, 변경 시 관리자 문의 |
| 연락처 | tel | - | 선택 입력 |
| 언어 | select | - | `한국어` / `English` |

### 알림 설정 카드 (`#notifications`)

| 알림 항목 | 기본값 |
|-----------|--------|
| 매체 오프라인 경보 | on |
| 편성 충돌 감지 | on |
| 캠페인 승인 요청 | on |
| 신규 사용자 합류 | on |
| 편성표 즉시 반영 완료 | off |
| 주간 리포트 이메일 | on |

### 보안 카드 (`#security`)

| 필드 | 타입 | 필수 | 비고 |
|------|------|------|------|
| 현재 비밀번호 | password | * | 눈 아이콘 토글 |
| 새 비밀번호 | password | * | 강도 표시 바 (weak/medium/strong), 눈 아이콘 토글 |
| 새 비밀번호 확인 | password | * | |

## 기타 UI

- **비밀번호 강도 바**: pw-weak / pw-medium / pw-strong (CSS 클래스)
- **저장 바**: 변경 사항 있을 때 하단 sticky bar 표시 (저장 / 취소)
- **Danger Zone 카드**: 계정 비활성화 등 위험 액션 (danger 스타일)
- **토스트**: 저장 완료 시 하단 토스트 표시
