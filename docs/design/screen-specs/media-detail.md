# 매체 상세 (`media-detail.html`)

## 기본 정보

- **페이지 제목**: {매체명}
- **브레드크럼**: 매체 관리 › 매체 목록 › {매체명}
- **레이아웃**: 사이드바(200px) + 메인 (GNB + 컨텐츠)

## 레이아웃 구조

- GNB: 브레드크럼 + 우측 날짜 + 알림벨
- 페이지 헤더: 매체명 + 상태 배지 + 활성/비활성 토글 + 수정 버튼
- 동기화 상태 바 (마지막 동기화 시각 표시)
- 탭 3개: 기본 정보 / 디바이스 연동 / 헬스체크 이력

## 탭별 컨텐츠

### 탭 1: 기본 정보
- 정보 그리드 (key-value): 매체사, 매체 종류, 설치 형태, 디스플레이 유형, 해상도, 화면 사이즈, LED Pitch, 운영 시간, 등록일, 비고

### 탭 2: 디바이스 연동
- 디바이스 ID (잠금 배지 포함), 연동일, 영상 출력 조정, 플레이어 버전, 마지막 통신

### 탭 3: 헬스체크 이력

#### 이력 필터
| 이름 | 타입 | 옵션 |
|------|------|------|
| 기간 | select | `7`→최근 7일, `14`→최근 14일, `30`→최근 30일 |

#### 헬스체크 테이블 컬럼
| # | 헤더 | 비고 |
|---|------|------|
| 1 | 시각 | datetime |
| 2 | 결과 | hc-ok / hc-warn / hc-err |
| 3 | 응답 시간 | ms 단위 |
| 4 | 세부 내용 | 텍스트 |

## 상태값/배지

```
// 헬스체크 결과
hc_result: 'hc-ok' | 'hc-warn' | 'hc-err'
```
