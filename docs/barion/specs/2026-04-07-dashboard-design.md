## 1. Anti-Factory GATE D1 체크

| 질문 | 답 |
|------|---|
| 화면 유형 | **모니터링** — 실시간 운영 현황 파악 |
| 주인공 | **이상 감지 건수** — 오프라인, 동기화 실패, 검수 실패 등 즉시 대응 필요 항목 |
| 도메인 특화 UI | 매체 위치 지도, 매칭 카드("의도 vs 현장"), 이상 감지 히어로 카드 |
| 기존 화면과 구조 겹침 | **의도적으로 다름** — 비대칭 그리드 + 3-zone 구조. 목록형 화면과 완전히 구분 |

## 3. 레이아웃 구조

```
┌──────────────────────────────────────────────────┐
│ GNB (공통)                                        │
├────────┬─────────────────────────────────────────┤
│        │ 페이지 헤더: 대시보드 / 날짜 / 기간필터 / 새로고침│
│ Side   │                                         │
│ bar    │ ┌─── ZONE 1: 공통 상단 (비대칭) ────────┐│
│        │ │ ┌──────────────┐ ┌────────────┐      ││
│        │ │ │              │ │ KPI: 매체   │      ││
│        │ │ │  이상 감지     │ ├────────────┤      ││
│        │ │ │  히어로 카드   │ │ KPI: 캠페인 │KPI:편성││
│        │ │ │  (2fr)       │ │ (1fr)      │(1fr) ││
│        │ │ └──────────────┘ └────────────┘      ││
│        │ └──────────────────────────────────────┘│
│        │                                         │
│        │ ┌─── ZONE 2: 공통 중단 ────────────────┐│
│        │ │ 최근 알림 (1fr)  │ 매체 위치 지도 (1fr)  ││
│        │ └──────────────────────────────────────┘│
│        │                                         │
│        │ ┌─── ZONE 3: 역할 전용 하단 ───────────┐│
│        │ │ (역할에 따라 내용 변경)                  ││
│        │ └──────────────────────────────────────┘│
└────────┴─────────────────────────────────────────┘
```

### ZONE 간 간격 원칙

- 카드 내부 요소: `--space-2` (8px)
- 같은 ZONE 내 카드 간: `--space-3` (12px)
- ZONE 간: `--space-6` (24px) — 시각적 호흡
- 비율: 1 : 1.5 : 3

## 4. 페이지 헤더

- 좌측: 페이지 제목 "대시보드" + 날짜 (YYYY년 M월 D일 요일)
- 우측: 기간 필터 + 새로고침 버튼
- **기간 필터**: select 형태. 옵션: 최근 7일(기본값) / 최근 30일 / 최근 90일
  - 적용 범위: KPI 카드 숫자, 최근 알림 목록에 영향. 이상 감지 히어로는 항상 실시간 (기간 필터 무관)
- **새로고침 버튼**: primary-500 텍스트 버튼. 클릭 시 전체 위젯 갱신 + "마지막 갱신 HH:mm" 업데이트
- 각 위젯 우측 상단에 "마지막 갱신: HH:mm" 표시 (text-xs, neutral-400)

## 5. ZONE 1 — 공통 상단 (비대칭 그리드)

### 구조

```
grid-template-columns: 2fr 1fr
```

좌측(2fr): 이상 감지 히어로 카드 (grid-row: span 2)
우측 상단(1fr): 전체 매체 KPI
우측 하단(1fr): 캠페인 + 편성 KPI (각 0.5fr)

### 이상 감지 히어로 카드

- **이상 있을 때 (1건 이상)**
  - 배경: error-50
  - 보더: 1.5px solid error-300
  - 그림자: `0 4px 12px rgba(error-500, 0.08)` — Level 2 부유 (shadow-md의 error tint 변형, 토큰 체계 외 커스텀값)
  - 상단: "⚠ 이상 감지" 라벨 + 건수 (28px+, weight 800, error-500)
  - 본문: 이상 매체 목록 (최대 5건). 각 항목에 매체명, 위치, 이상 유형 뱃지
  - 우측 상단: "전체 보기 →" 링크 → S-04 매체 목록 (이상 필터)

- **이상 없을 때 (0건)**
  - 같은 자리, 같은 크기
  - 배경: neutral-50 (바닥색과 동일하게 후퇴)
  - 보더: 1px solid border-default
  - 내용 전환: **"오늘의 요약"**
    - 오늘 예정된 편성 변경
    - 검수 대기 소재
    - 만료 예정 캠페인 (7일 이내)
    - 최근 24시간 이벤트 요약

### KPI 카드

| 카드 | 숫자 크기 | weight | 하위 정보 |
|------|----------|--------|----------|
| 전체 매체 | text-xl (22px) | 800 | "온라인 N · 오프라인 N · 비활성 N" (text-xs, 상태별 건수 모두 표시) |
| 캠페인 | text-md (16px) | 700 | "초안 N · 집행중 N · 완료 N · 취소 N" (text-xs) |
| 편성 | text-md (16px) | 700 | "예약 N · 적용중 N · 일시정지 N · 종료 N" (text-xs) |

- 배경: white (대시보드 Level 1 전용 — foundation의 bg-elevated를 카드 표면에 사용. bg-surface(neutral-100)보다 밝아야 비대칭 그리드에서 바닥과 구분됨)
- 보더: 1px solid border-default
- 그림자: shadow-sm — Level 1 표면
- 클릭 시 해당 목록 화면으로 이동

## 6. ZONE 2 — 공통 중단

### 구조

```
grid-template-columns: 1fr 1fr
```

### 좌측: 최근 알림

- 최대 5건 표시
- 각 항목: severity dot + 알림 텍스트 + 시각
- severity dot 형태 분화 (WCAG 1.4.1):
  - 긴급(error): 삼각형, error-500
  - 일반(normal): 원형, primary-500
  - 정보(info): 사각형(radius 2px), neutral-300
- 하단: "전체 보기 →" → S-24 알림 센터

### 우측: 매체 위치 지도

- 위경도 기반 매체 위치 시각화
- 매체 상태별 핀 색상: 온라인=primary-500, 오프라인=error-500, 비활성=neutral-300
- 상호작용: 핀 클릭 → 매체 상세(S-04)로 이동
- 빈 상태: "등록된 매체가 없습니다" + 매체 등록 링크

## 7. ZONE 3 — 역할 전용 하단

### 어드민

```
grid-template-columns: 1fr 1fr 1fr
```

| 위젯 | 주 정보 | 부 정보 |
|------|--------|--------|
| 동기화 미완료 | 건수 (text-lg, warning-500 if >0) | 매체명 나열 |
| 배치 실행 상태 | "정상" 또는 "이상" | 마지막 실행 시각 |
| Slack 연동 | "정상" 또는 "실패" | 마지막 발송 시각 |

비정상 캠페인(시작일 경과 + 초안 상태)이 있으면 하단에 경고 카드 추가.

### 매체사 — 동기화 현황 (B+C 혼합)

**불일치 있을 때:**

매칭 카드 형태 — 매체별로 "CMS 의도"와 "플레이어 현장"을 좌우 대비.

```
┌─────────────────────────────────────────┐
│ 강남역 1번출구                  [불일치]  │
│ ┌──────────────┐     ┌──────────────┐  │
│ │ CMS 의도      │ ≠  │ 플레이어 현장  │  │
│ │ 4월 프로모션   │     │ v2.1 미반영   │  │
│ │ 적용중        │     │              │  │
│ └──────────────┘     └──────────────┘  │
└─────────────────────────────────────────┘
```

- 배경: error-50, 보더: error-300
- "≠" 기호로 불일치 시각화

**정상 매체:**

한 줄 축약 — "🟢 정상 동기화 3개 매체: 종로 전광판, 삼성역 B2, 역삼역 3번출구"

- 배경: neutral-50, 보더: border-default

**전체 정상 (불일치 0건):**

매칭 카드 없이 축약만 표시.

### 운영대행사

- 담당 매체사 컨텍스트 스위처 (segment control)
- 선택한 매체사 기준으로 ZONE 1~2 데이터 필터
- 동시편집 경고:
  - 트리거: 다른 사용자가 같은 매체사의 편성을 편집 중일 때 (WebSocket 또는 polling으로 감지)
  - 위치: ZONE 3 최상단
  - 스타일: warning-50 배경, warning-200 보더, text-sm
  - 텍스트: "⚠ [사용자명]님이 [매체사명]의 편성을 편집 중입니다"
  - 해제: 해당 사용자가 편집을 종료하면 자동 해제

## 8. 비주얼 해석 가이드 (v2 — 시각적 에너지 적용)

design-foundation.md 토큰을 이 화면에서 어떻게 쓰는지. anti-factory 원칙 6(표면의 시각적 에너지) 반영.

### 컬러 블록 전략

**사이드바:** 다크 그린 (primary-900 ~ oklch(0.30 0.07 155)) 배경. 아이콘/텍스트 white. active 항목은 primary-500으로 하이라이트.

**이상 감지 히어로:** 다크 그린 배경 + white 텍스트. 이상 0건이면 neutral-100 배경으로 전환.

| 영역 | 배경색 | 텍스트색 |
|------|--------|---------|
| 사이드바 | primary-900 (oklch(0.30 0.07 155)) | white, active=primary-400 |
| 이상 감지 히어로 (이상 있음) | primary-900 (oklch(0.30 0.07 155)) | white, 이상 뱃지=error tint |
| 이상 감지 히어로 (이상 없음) | neutral-100 | neutral-900 |
| KPI 카드 | white | neutral-900 |
| 페이지 배경 | neutral-50 | — |

### 컬러 — 경보 등급 (유지)

| 색 | 의미 | 사용처 |
|----|------|-------|
| error-500 (red) | 즉시 대응 | 오프라인, 동기화 실패, 검수 실패 |
| warning-500 (yellow) | 주의 관찰 | GPU 온도, 검수 대기, 비정상 캠페인 |
| neutral (gray) | 정상/배경 | KPI 숫자, 정상 매체, 일반 텍스트 |
| primary-500 (green) | 행동 유도만 | 새로고침 버튼, "전체 보기" 링크 |

### 데이터 시각화

시각화는 "제거하면 정보 전달이 나빠지는가?" Yes인 경우에만.

| 위젯 | 시각화 | 이유 |
|------|--------|------|
| 이상 감지 | 미니 바 차트 (최근 7일 추이) | "오늘만 3건인지, 추세가 있는지" 판단 |
| 전체 매체 | 도넛 차트 | 온라인/오프라인/비활성 비율이 숫자보다 직관적 |
| 캠페인 | 프로그레스 바 | 상태별 비율을 한 줄로 표현 |
| 편성 | 프로그레스 바 | 동일 |
| 최근 알림 | ❌ 텍스트 유지 | 내용을 읽어야 하므로 차트 불가 |
| 매체 위치 | ❌ 지도 유지 | 이미 도메인 특화 |
| 어드민 카드 | ❌ 텍스트 유지 | 이진 상태(정상/이상)에 차트 불필요 |

### 타이포 — 숫자 무게 3단계 (유지)

| 역할 | 크기 | weight | 용도 |
|------|------|--------|------|
| 주인공 | text-2xl (28px)+ | 800 | 이상 감지 건수 (white on dark) |
| 조연 | text-xl (22px) | 800 | 전체 매체 수 (도넛 중앙) |
| 배경 | text-md (16px) | 700 | 캠페인, 편성 수 |
| 라벨 | text-xs (11px) | 500 | 카드 제목, 부가 정보 |

공통: `letter-spacing: -0.03em`, `font-variant-numeric: tabular-nums`

### Elevation — 3단계 (수정)

| 레벨 | 용도 | 스타일 |
|------|------|--------|
| 0 바닥 | 페이지 배경 | neutral-50, 그림자 없음 |
| 1 표면 | KPI 카드, 알림 패널, 지도 | white, border-default, shadow-sm |
| 2 강조 | 사이드바, 이상 감지 히어로 | primary-900 다크 배경, 컬러 면적으로 elevation 표현 |
