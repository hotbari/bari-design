# 박수현 — 8년차 시니어

> "나도 가끔 조항 번호 찾는 게 귀찮아."

## Demographics
- 나이: 35세
- 직무: 구매/법무 지원팀 8년차
- 기술 수준: 사내 규정 숙지도 높음, 후배 질문 자주 받음

## Goals
- **Functional:** 기억 애매한 세부 조항을 빠르게 확인
- **Social:** 후배 질문에 근거 있게 정확하게 답해주기

## Frustrations
- 아는 것 같은데 정확한 수치/날짜/조항 번호가 기억 안 남
- 후배한테 틀린 답변을 주는 게 가장 싫음
- 직접 문서 찾아주면 시간이 너무 걸림

## Behaviors
- 기억으로 먼저 답하고, 나중에 문서로 확인
- 중요한 계약/검토 건에는 원문 반드시 확인

## Design Implications
- 출처(문서명+페이지) 표시가 핵심 — 신뢰 근거
- 답변이 짧고 정확해야 함. 긴 설명 불필요
- 검토 요청 시 주의사항 선제 지적으로 전문가처럼 느끼게
