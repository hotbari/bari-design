# 인터뷰 인사이트 — 김서영 (매체사 운영팀장)

**일시**: 2026-04-03
**페르소나**: 김서영 (34세, 시티미디어 운영팀장, 매체사)
**방식**: 페르소나 시뮬레이션 인터뷰
**목적**: 시나리오 검증 미결 사항에 대한 실무 관점 수집

---

## 페르소나 정의와 다른 응답

- **매체 범위 정정**: 페르소나에 "버스정류장 사이니지 200대, 지하철역 50대"로 기재 → 실제로는 옥외 광고판(빌딩 외벽, 도로변 전광판)만 해당. 페르소나 파일 수정 필요
- **광고주 관리 현실**: CMS 드롭다운 선택이 아닌 엑셀/구글 시트로 별도 관리 중. CMS 내 광고주 데이터 품질 낮음 (중복 등록, 명칭 불일치)

## 검증된 가설

| 가설 | 검증 결과 | 관련 |
|------|----------|------|
| 매체 삭제보다 "보관" 필요 | 확인. 이력 보존 + 기본 목록 제외 원함 | 02 미결 |
| 디바이스 교체 시나리오 누락 | 확인. DB 직접 수정으로 우회, 블랙스크린 발생 경험 | 02 미결 |
| 소재 만료 시 경고만 (C-13) | 확인. 재계약 케이스로 자동 제거 불가. 만료 임박 알림 추가 요청 | C-13 해소 |
| 편성 캘린더뷰 필요 | 확인. 엑셀 간트 차트 병행 관리 중 | 05 미결 |
| 네비게이션 과부하 | 확인. "기능 기준" → "업무 기준" 또는 즐겨찾기 요청 | 공통 UX |

## 신규 발견

### I-1. 집행중 캠페인 매체 변경 우회 패턴 [중]
- **현상**: 매체 추가/제거 시 기존 캠페인을 놔두고 새 캠페인을 생성하여 우회
- **문제**: 정산 시 동일 건이 2건으로 분리되어 혼란
- **관련**: 04 캠페인 미결 "집행중 캠페인 수정 가능 범위"
- **제안**: 집행중 캠페인에서 수정 가능 필드를 명시 — 최소한 대상 매체 추가/제거, 종료일 변경 허용

### I-2. 캠페인 종료일 단축 시 편성표 연쇄 처리 [중]
- **현상**: 종료일을 당기면 편성표가 어떻게 되는지 불명확
- **문제**: C-10에서 캠페인 "취소" 시 연쇄 종료는 확정했으나, "종료일 단축"은 취소가 아님
- **관련**: 04↔05 교차
- **제안**: 캠페인 종료일 변경 시 연결 편성표의 적용 기간을 초과하는 경우 경고 + 편성표 종료일 동기화 여부 확인

### I-3. 소재 만료 임박 알림 [하]
- **현상**: 종료일 직전까지 인지 못하고 만료 소재가 계속 송출
- **제안**: 종료일 N일 전 (기본 3일) 알림 발송. 06 이벤트 카탈로그에 추가

### I-4. 광고주 중복 등록 방지 [중]
- **현상**: "삼성전자", "삼성전자(주)", "(주)삼성전자"가 별도 등록
- **제안**: 사업자등록번호 유니크 키. 광고주 등록 시 중복 체크 필수
- **필수 속성**: 회사명, 사업자등록번호, 담당자명, 연락처

### I-5. 매체 "보관(아카이브)" 상태 [중]
- **현상**: 철거/계약종료 매체가 비활성으로만 남아 목록 오염
- **제안**: 매체 FSM에 "보관됨" 상태 추가. 기본 목록 비표시, 이력 조회 전용
- **연쇄 효과**: 보관된 매체의 소속 매체사 삭제 가능 여부도 정의 필요

### I-6. 현장 확인 기능 (원격 스크린샷) [중]
- **현상**: 편성 반영 후 현장 확인을 운영대행사 사진 촬영에 의존
- **제안**: 플레이어에서 주기적 스크린샷 → CMS 표시. UC-11 헬스체크 확장
- **비고**: 기술적 실현 가능성은 플레이어 팀 확인 필요

### I-7. 대시보드 지도에 네이버 지도 선호 [하]
- **근거**: 주소 검색 정확도, 거래처 위치 공유 관행
- **관련**: 07 UC-29 매체 위치 지도 위젯

### I-8. 편성 캘린더뷰 + 실시간 충돌 시각화 [중]
- **현상**: 편성 목록 표 형태 → 시간축 파악 불가, 엑셀 병행
- **제안**: 매체별 편성 타임라인 (간트 차트). 충돌 구간 색상 표시. 드래그 중 실시간 충돌 감지
- **관련**: 05 미결 "편성 타임라인 시각화"

### I-9. 자주 쓰는 기능 즐겨찾기 [하]
- **현상**: 편성 관리, 소재 관리, 대시보드를 하루 수십 회 왕복
- **제안**: 사용자별 즐겨찾기 또는 대시보드 바로가기 위젯

## 추가 리서치 필요 영역

- 현장 확인(원격 스크린샷) 기술적 실현 가능성 → 플레이어 팀
- 캠페인 금액 필드 활용도 → "메모 수준"이면 필수→선택 전환 검토
- 매체 보관 상태에서 리포트 조회 범위 → 보관 매체 데이터 노출 범위
- 운영대행사(이준혁) 인터뷰로 현장 작업 관점 보완 필요
